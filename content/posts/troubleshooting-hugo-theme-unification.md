# 증상

메인 화면에서는 브랜드 인상이 분명했지만, `Posts`나 `Projects`로 이동하면 검은 배경 위에 기본 텍스트만 보였다.

# 확인한 점

- 홈은 React 번들에서 별도로 렌더링되고 있었다.
- 내부 페이지는 Hugo의 기본 PaperMod 템플릿이 출력되고 있었다.
- 두 영역이 같은 데이터는 보지만 같은 레이아웃 시스템을 공유하지는 않았다.

# 해결

Hugo 쪽 `header`, `footer`, `single`, `list` 템플릿을 오버라이드해서 홈과 비슷한 유리 질감 카드, 다크 톤, 타이포그래피를 공통으로 사용하게 만들었다.

# 결과

이제 홈에서 내부 페이지로 이동해도 시각 언어가 끊기지 않고, 새 분류 페이지도 같은 구조 안에서 확장할 수 있다.
