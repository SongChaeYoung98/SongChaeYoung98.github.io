# 작업 배경

상단 메뉴에는 글 성격에 맞는 이동 경로가 없었고, 메인 화면의 일부 버튼도 실제 페이지로 연결되지 않았다.

# 이번 변경

- 홈 상단 메뉴에 분류형 링크 추가
- `troubleshooting`, `architecture`, `devlog` 허브 페이지 구현
- 내부 페이지 공통 스타일을 랜딩과 비슷한 톤으로 통일

# 남긴 메모

이 구조는 이후 새 글을 쓸 때 태그나 카테고리만 맞춰주면 자동으로 허브에 노출된다는 점이 가장 편하다.
